## [1.0.0] - 2026-01-17

### Added
- Initial MVP release of Drift Clock
- Timer with numpad-style input (00h 00m 00s format)
- Stopwatch with lap tracking
- Time manipulation engine with 5 speed profiles:
  - Turbo (1.25x faster)
  - Grind (0.8x slower)
  - Stealth (1.1x faster)
  - Trickster (1.5x faster)
  - Custom (0.5x - 2.0x adjustable)
- Hidden settings accessible via Bedtime tab long-press
- Adaptive sizing for different screen sizes
- Material Design 3 theming matching stock Android Clock
