# Feature Roadmap: Drift Clock

## MVP (Phase 1)

### Core UI
- [ ] 5-tab bottom navigation matching stock Android Clock
  - Alarm (decorative/fake)
  - Clock (decorative/fake)
  - Timer (functional with speed manipulation)
  - Stopwatch (functional with speed manipulation)
  - Bedtime (decorative + hidden settings access)

### Profile System
- [ ] Full-screen profile selection on first launch
- [ ] 5 preset profiles:
  - **Turbo** (1.25x) - "Rest periods fly by. That 60-second break? Done in 48."
  - **Grind** (0.8x) - "Make every second burn. Your 30-second plank just became 36."
  - **Stealth** (1.1x) - "Subtle enough to fool yourself. Perfect for pushing through study sessions."
  - **Trickster** (1.5x) - "For party bets and bar tricks. 'Watch me hold my breath for 2 minutes.'"
  - **Custom** (user-defined) - "Dial in your own time distortion. Fine-tune to the decimal."
- [ ] Profile persistence (remember selection)

### Timer Tab (Functional)
- [ ] Stock Android Clock Timer UI clone
- [ ] Time input (hours, minutes, seconds)
- [ ] Start, pause, reset controls
- [ ] Time runs at selected profile speed
- [ ] Local notification on completion
- [ ] Sound/vibration alert

### Stopwatch Tab (Functional)
- [ ] Stock Android Clock Stopwatch UI clone
- [ ] Start, pause, reset controls
- [ ] Lap functionality
- [ ] Time runs at selected profile speed

### Hidden Settings
- [ ] Access via Bedtime tab (hidden gesture or long-press)
- [ ] Change active profile
- [ ] Custom speed slider/input
- [ ] Reset to defaults

### Fake Tabs (Decorative)
- [ ] Alarm tab - static UI that looks real
- [ ] Clock tab - shows current time (real time)
- [ ] Bedtime tab - static UI with hidden settings trigger
